# 초등임용 교육과정 비교 프로젝트 하네스

이 프로젝트의 1차 사용자는 초등임용고시를 준비하는 교대생·수험생이다. 목표는 교육과정 문구를 단순 검색하는 것이 아니라, 2015·2022 개정 교육과정의 변화가 기출문제의 수업 장면, 수업 설계, 평가 증거와 어떻게 연결되는지 인출 가능하게 돕는 것이다.

## 핵심 줄기

1. 교육과정·평가기준·성취수준 PDF를 OCR/텍스트 추출한다.
2. 2015·2022 성취기준을 교과·학년군·영역 단위로 추출한다.
3. 성취기준 대응표를 생성한다.
   - `unchanged`: 코드와 문구가 거의 유지됨
   - `same_code_modified`: 같은 코드이나 문구가 수정됨
   - `semantic_match`: 코드가 달라도 의미상 대응되는 것으로 자동 추정됨
   - `removed_or_unmatched`: 2015에는 있으나 2022에서 직접 대응을 찾지 못함
   - `new_or_unmatched`: 2022에 새로 나타나거나 2015 대응을 찾지 못함
4. 교육과정 원문, 평가기준·성취수준, 기출문제, 대응표를 하나의 벡터DB로 재구축한다.
5. Streamlit/FastAPI에서 교과·학년군·영역 필터로 비교 검색한다.
6. 답변은 초등임용 수험생 관점에서 다음 순서로 구성한다.
   - 핵심 변화
   - 수업 설계에 미치는 영향
   - 관련 기출 관점
   - 인출 질문

## 현재 의도적으로 보류한 것

2015년 기출문제 OCR은 일반 파서가 실패하므로 별도 보정 파서를 사용한다. OCR 텍스트가 거칠 수 있어 원문 완전 복원 자료가 아니라 기출 관점 검색용 chunk로 취급한다.

앱과 답변에서는 다음 제한을 유지한다.

- 2009 개정 교육과정은 비교 범위에서 제외한다.
- 기출 정답·모범답안이 없는 상태이므로 공식 정답처럼 단정하지 않는다.
- 기출 답안은 필요한 경우 `근거 기반 예상 답안`으로만 표현한다.
- `semantic_match`, `removed_or_unmatched`, `new_or_unmatched`는 자동 추정이므로 검수 필요 항목으로 본다.

## 현재 검증 결과

- OCR 배치: 36개 성공, 0개 실패, 중복 1개
- 성취기준 추출: 2015년 554개, 2022년 612개
- 대응표: 775개
- 검수 필요 대응: 389개
- 2015 기출 OCR 보정: A형 11문항, B형 11문항
- 벡터DB: 8,407개 문서 청크/기출/대응표
- 기본 대표 질문 검증: 7개 체크 모두 통과
- 교과별 대표 질문 검증: 30개 체크 모두 통과
- Streamlit UI 하네스: 4개 체크 모두 통과
- 자동 대응표 검수 큐: 389개 중 high 1개, medium 47개, low 341개

대표 질문 검증 범위:

- 국어 5-6 토의·토론
- 수학 5-6 분수의 나눗셈
- 과학 3-4 소리의 성질
- 통합교과 1-2 학교와 생활
- 과정 중심 평가

추가 대표 질문 검증은 국어, 수학, 과학, 사회, 도덕, 실과, 체육, 음악, 미술, 영어, 통합교과 및 평가·수업 설계 공통 질문 30개로 구성한다.

## 하네스 안전장치

- OCR 결과는 캐시되어 반복 실행 병목을 줄인다.
- 비교표가 없으면 필터와 비교 결과는 빈 값으로 안전하게 반환한다.
- OpenAI 답변 생성이 실패해도 `/comparison` API는 비교표와 관련 기출을 반환한다.
- 관련 기출 검색은 벡터DB가 있을 때만 수행한다.
- 자동 대응표의 추정 항목은 항상 검수 필요로 표시한다.
